# 合力牧业农牧乳一体化经营管理平台

当前项目定位为“合力牧业农牧乳一体化经营管理平台前端演示版”。系统从单一牧场管理升级为集团化业务平台，覆盖奶牛养殖、肉牛育肥、饲料生产、原奶入厂、乳品加工、物流配送和财务经营。

最新信息架构已从“功能页面堆叠”优化为“工作模式 + 工作台 + 二级明细页”：

- 工作模式：经营总览、业务操作、平台配置
- 上层入口：集团驾驶舱、四大业务板块工作台、平台能力总览
- 二级明细页：具体台账、记录、库存、审批、事件等仍保留原路由，但主要从工作台进入

业务主体名称统一为：

- 奶牛场：合力牧业奶牛场
- 肉牛场：欧力菲德肉牛场
- 饲料厂：欧力菲德饲料厂
- 乳品厂：欧力菲德乳品厂

## 核心模块

- 经营总览：首页驾驶舱、数据大屏、经营报表、AI 经营助手、平台能力总览
- 四大板块工作台：合力牧业奶牛场工作台、欧力菲德肉牛场工作台、欧力菲德饲料厂工作台、欧力菲德乳品厂工作台
- 统一协同：工单中心、牛只查询中心、质量追溯中心
- 平台能力：消息预警中心、审批中心、角色组织模拟、牛只事件时间线、自定义分析、接口管理中心、规则中心、操作日志、员工工作台、知识库手册、系统建设方案
- 合力牧业奶牛场：奶牛档案、牛舍管理、智能分群、饲喂计划、饲喂记录、繁育管理、健康防疫、产奶记录、原奶配送、奶牛场库存
- 欧力菲德肉牛场：肉牛档案、育肥批次、体重记录、饲喂执行、出栏销售、生物资产台账、肉牛场库存
- 欧力菲德饲料厂：原料库存、饲料成品、饲料生产、内部调拨、采购记录
- 欧力菲德乳品厂：原奶接收、质检记录、生产计划、生产批次、灌装记录、成品质检、成品入库、成品出库、退货记录、成品库存、销售订单
- 物流车辆：配送任务、货车管理、地磅过磅
- 设备数据：设备状态中心、牛舍环境监控
- 客商员工：客户供应商、员工岗位
- 财务经营：收支账本、成本统计、利润分析
- 系统管理：公告通知、基础设置、数据导入导出

左侧菜单会根据当前工作模式收敛展示：

- 经营总览：看集团驾驶舱、数据大屏、经营报表、AI、四大业务工作台、消息和审批。
- 业务操作：处理员工工作台、工单、产奶、饲喂、质检、配送、入库出库和过磅。
- 平台配置：展示角色组织、接口、规则、分析、知识库、操作日志、系统设置和建设方案。

角色视图会影响菜单和默认入口：

- 奶牛场场长默认进入合力牧业奶牛场工作台。
- 肉牛场场长默认进入欧力菲德肉牛场工作台。
- 饲料厂厂长默认进入欧力菲德饲料厂工作台。
- 乳品厂厂长默认进入欧力菲德乳品厂工作台。
- 普通员工默认进入员工工作台。
- 只读访客隐藏新增、编辑、删除类入口，仅保留查看类页面。

平台通过 mock 数据展示“业务流、物流、资金流、数据流四流合一”：工单驱动业务执行，配送与过磅体现物流，账本和生物资产体现资金，设备、环境、追溯和 AI 简报体现数据流。

最新版本进一步扩展为“业务流、物流、资金流、数据流、权限流、消息流、设备流”综合管理：

- 消息流：统一消息中心承接系统通知、工单、库存、设备、环境、防疫、繁育和审批提醒。
- 权限流：角色视图模拟集团管理员、场长、厂长、物流、财务、司机和只读访客的菜单范围。
- 数据流：规则中心、自定义分析、接口管理中心和操作日志体现数据治理能力。
- 设备流：设备状态、环境监控、地磅过磅和外部接口配置展示未来 IoT 接入路径。

## 业务联动示例

- 欧力菲德饲料厂生产奶牛精补料和育肥牛饲料。
- 欧力菲德饲料厂向合力牧业奶牛场调拨奶牛精补料。
- 欧力菲德饲料厂向欧力菲德肉牛场调拨育肥牛饲料。
- 合力牧业奶牛场产出的原奶配送到欧力菲德乳品厂。
- 欧力菲德乳品厂接收原奶后进行质检、生产巴氏鲜奶和酸奶，并形成成品库存和销售订单。
- 欧力菲德肉牛场记录育肥批次、体重和出栏销售。
- 财务账本按“合力牧业奶牛场、欧力菲德肉牛场、欧力菲德饲料厂、欧力菲德乳品厂、集团”统计收入、支出和利润。

## 数据保存

系统仍为纯前端演示版，所有数据保存在浏览器 localStorage：

```text
heli-smart-pasture-platform-v1
```

系统设置页支持导出 JSON、导入 JSON、重置示例数据和清空本机数据。导出数据包含以下新增平台级结构：

- `messages`
- `alertRules`
- `approvalRequests`
- `operationLogs`
- `cowEvents`
- `analysisTemplates`
- `externalInterfaces`
- `knowledgeBase`
- `roleViews`
- `mobileTasks`

点击首页或系统设置中的“重置示例数据”，会恢复包括上述结构在内的完整 mock 示例。

## 前端模拟边界

当前不接入真实后端、真实数据库、真实登录、真实 AI API、真实 IoT、真实冷链采集、真实审批流、真实短信 / 微信推送、真实企业微信、真实小程序、真实发票系统或真实财务对账。审批、接口测试、角色权限、审计日志、规则扫描和 AI 问答均为前端模拟，用于展示后续企业级系统架构方向。

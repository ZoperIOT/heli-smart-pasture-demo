# 合力牧业农牧乳一体化经营管理平台

当前项目是“合力牧业农牧乳一体化经营管理平台前端演示版”。系统用于向牧场企业、乳品加工厂和农牧数字化厂商展示：一个集团化农牧乳平台如何把奶牛场、肉牛场、饲料厂、乳品厂、配送车辆、财务账本、报表分析、消息协同和权限视图组织到同一套经营后台中。

项目保持纯前端演示：不接真实后端、数据库、IoT 设备、AI API、企业微信、小程序、金融系统或发票系统。所有业务数据均为 mock 数据，并保存在浏览器 localStorage。

## 业务主体

- 总平台：合力牧业农牧乳一体化经营管理平台
- 奶牛场：合力牧业奶牛场
- 肉牛场：欧力菲德肉牛场
- 饲料厂：欧力菲德饲料厂
- 乳品厂：欧力菲德乳品厂

## 信息架构

系统已从“功能页面堆叠”调整为“真实企业后台 + 角色工作台 + 五流合一 + 业务闭环 + 报表分析 + 消息协同”：

- 经营总览：首页驾驶舱、数据大屏、经营报表、AI 经营助手、平台能力总览。
- 业务流：四大板块工作台、工单、产奶、饲喂、质检、配送、过磅、入库出库、销售订单。
- 资金流：收支账本、成本统计、利润分析、应收应付、经营报表和审批。
- 数据流：追溯中心、自定义分析、操作日志、外部接口、设备状态示例和环境监控示例。
- 消息流：消息预警、工单提醒、审批提醒、员工移动工作台。
- 权限与配置：角色组织、基础设置、规则中心、知识库、系统建设方案、导入导出。

这里的“五流合一”指业务流、资金流、数据流、消息流、权限流。配送、过磅、车辆等物流能力归入业务流；设备和环境监控只作为未来 IoT 接入能力的前端示例，不作为独立主流程。

## 核心功能

- 集团经营驾驶舱：展示四大板块关键指标、今日工单、异常提醒、经营收入支出和最近动态。
- 四大业务工作台：分别服务合力牧业奶牛场、欧力菲德肉牛场、欧力菲德饲料厂、欧力菲德乳品厂。
- 工单中心：统一管理饲喂、产奶、防疫、配送、维修、质检、盘点和异常处理任务。
- 牛只查询中心：按耳标、牛舍、状态、重点关注和所属板块快速查牛。
- 饲喂与分群：展示日粮配方、饲喂计划、饲喂执行、智能分群、料奶比和料肉比。
- 肉牛经营：育肥批次、体重记录、增重分析、出栏销售、生物资产台账。
- 奶牛经营：奶牛档案、牛舍管理、繁育管理、健康防疫、产奶记录、原奶配送。
- 饲料厂经营：原料库存、成品库存、饲料生产、内部调拨、采购记录。
- 乳品厂经营：原奶接收、质检记录、生产计划、生产批次、灌装、成品质检、成品出入库、退货和销售订单。
- 质量追溯：展示饲料原料批次、饲料生产、饲喂、原奶/肉牛批次、乳品生产和销售订单的链路。
- 报表分析：按集团和四大板块查看收入、支出、库存、生产、异常、应收应付和利润。
- AI 经营助手：纯前端预设回答，模拟老板用自然语言查询经营情况。
- 角色视图：模拟集团管理员、场长、厂长、物流主管、财务人员、普通员工、司机和只读访客。

## 业务联动示例

- 欧力菲德饲料厂生产奶牛精补料和育肥牛饲料。
- 欧力菲德饲料厂向合力牧业奶牛场调拨奶牛精补料。
- 欧力菲德饲料厂向欧力菲德肉牛场调拨育肥牛饲料。
- 合力牧业奶牛场产出的原奶配送到欧力菲德乳品厂。
- 欧力菲德乳品厂接收原奶后进行质检、加工、成品入库和销售。
- 欧力菲德肉牛场记录育肥批次、体重变化和出栏销售收入。
- 财务账本按合力牧业奶牛场、欧力菲德肉牛场、欧力菲德饲料厂、欧力菲德乳品厂、集团统计收入、支出和利润。

## 前端模拟边界

当前不接入真实后端、真实数据库、真实登录、真实 AI API、真实 IoT、真实冷链采集、真实审批流、真实短信/微信推送、真实企业微信、真实小程序、真实发票系统或真实财务对账。审批、接口测试、角色权限、审计日志、规则扫描、消息联动和 AI 问答均为前端模拟，用于展示后续企业级系统架构方向。

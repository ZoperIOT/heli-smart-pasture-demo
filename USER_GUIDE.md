# 合力牧业农牧乳一体化经营管理平台使用说明

## 1. 集团首页

打开系统后进入“首页驾驶舱”。首页展示合力牧业奶牛场、欧力菲德肉牛场、欧力菲德饲料厂、欧力菲德乳品厂的核心指标，包括奶牛存栏、肉牛存栏、今日产奶、原奶入厂、饲料厂产量、乳品成品库存、配送任务、收支和异常提醒。

## 2. 合力牧业奶牛场

进入“合力牧业奶牛场”菜单，可以查看奶牛档案、牛舍管理、产奶记录、原奶配送和奶牛场库存。

产奶记录用于记录早班、晚班和总产奶量。原奶配送任务用于展示合力牧业奶牛场向欧力菲德乳品厂配送原奶。

## 3. 欧力菲德肉牛场

进入“欧力菲德肉牛场”菜单，可以查看肉牛档案、育肥批次、体重记录、出栏销售和肉牛场库存。

出栏销售记录可用于展示肉牛销售收入，并在财务账本中按“欧力菲德肉牛场”统计。

## 4. 欧力菲德饲料厂

进入“欧力菲德饲料厂”菜单，可以查看原料库存、饲料成品、饲料生产、内部调拨和采购记录。

内部调拨用于展示欧力菲德饲料厂向合力牧业奶牛场调拨奶牛精补料，向欧力菲德肉牛场调拨育肥牛饲料。

## 5. 欧力菲德乳品厂

进入“欧力菲德乳品厂”菜单，可以查看原奶接收、质检记录、生产批次、成品库存和销售订单。

原奶接收示例来自合力牧业奶牛场。生产批次展示巴氏鲜奶、酸奶等成品加工。销售订单展示乳品销售和后续配送能力。

## 6. 物流车辆

进入“物流车辆 / 配送任务”，可以查看原奶配送、饲料调拨、乳品配送等任务。任务改为“配送中”时货车同步为配送中；任务改为“已到达”时货车恢复为空闲。

## 7. 财务经营

进入“财务经营 / 收支账本”，可以按业务板块筛选账本记录。业务板块包括：

- 合力牧业奶牛场
- 欧力菲德肉牛场
- 欧力菲德饲料厂
- 欧力菲德乳品厂
- 集团

经营报表和利润分析会按板块展示收入、支出和利润。

## 8. 数据大屏和经营报表

数据大屏用于展示四大板块运行状态、今日产奶、肉牛存栏、饲料厂产量、乳品销售额、物流配送、库存预警和 AI 简报。

经营报表展示分板块收入、成本、利润，以及产奶、配送、库存、车辆和收支图表。

## 9. AI 经营助手

AI 经营助手为前端本地模拟，不调用真实大模型。它会根据当前 localStorage 数据回答：

- 今天集团整体经营情况怎么样？
- 奶牛场今天产奶多少？
- 肉牛场当前存栏多少？
- 饲料厂有哪些库存预警？
- 乳品厂今天接收了多少原奶？
- 本月哪个板块收入最高？
- 有哪些配送任务还没完成？
- 给出本周经营建议。

## 10. 系统管理

公告通知用于发布集团或业务板块通知。基础设置用于修改平台名称、负责人、电话、默认奶价、默认记录人等。

数据导入导出位于系统设置页，可导出 JSON 备份、导入 JSON 恢复、重置示例数据或清空本机数据。

## 11. 本地和手机访问

```powershell
cd E:\Projects\JAVA\heli-smart-pasture-demo
npm run dev -- --host 0.0.0.0
```

电脑访问：

```text
http://127.0.0.1:5173
```

手机与电脑连接同一 Wi-Fi 后，在 Safari 输入：

```text
http://电脑IP:5173
```

## 12. 数据说明

当前为静态前端演示版，所有数据保存在当前浏览器 localStorage 中。刷新页面不会丢失；更换浏览器或设备不会自动同步。
